Importing SmartSection content:

1. Login as administrator and install Impression.
2. Make sure that the following tables exist in your database:
   - smartsection_categories
   - smartsection_items
3. To start the import access the import script: <website_url>/modules/impression/import/smartsection.php
4. When the import is finished the text "Done!" will be displayed on the screen
5. Remove the folder <website_url>/modules/impression/import from your server
6. You should be able to use Impression and have most important content from SmartSection
